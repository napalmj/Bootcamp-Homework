-Flask Shazam Api Song Search Site-
-----------------------------------

Author: Nathaniel Palmer
Version Python: 3.9.5
Using Flask, Jinja 2.

-----------------------------------
Functionality:

User UI for asking user for song.
-->Provides user with song art and link to more recommended.
-->shows recommended songs with title and song art

-----------------------------------
Start up:

In root directory --> execute follow bash commands:

-export FLASK_APP=song-server-back.py
-export FLASK_APP=development
-start flask "or" python3 -m flask run

Follow the directions in terminal window of which local port to view the hosted site.
