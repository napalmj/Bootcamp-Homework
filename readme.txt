---------------------------------------------------
-----Song Recommender Script using Shazam api------
---------------------------------------------------

Version Python: 3.9.5
Operating System: macOS Big Sur v11.4
Author: Nathaniel Palmer
Api Endpoints Used: /search, /songs/list-recommendations


----------------------------------------------------
---------Script Usage and Set Up--------------------
----------------------------------------------------

Script Usage:-----------

->Takes in song from user. 
-->Informs user of 3 songs relating to their choice
-->Asks for them to choose which they prefer
->Provides song recommendations based off of their choice


Set Up:-----------------

->Download repository zip or clone project via git
->run: 'python3 file-name.py'
->Fill in the script promps

Note: you will need to be connected to the internet.
